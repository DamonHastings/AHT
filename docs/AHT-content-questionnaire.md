## How to use this

- Answer **in order** when you can. Short lines work well for headlines; **1–3 sentences** per prompt is enough unless noted.
- If something does not apply, write **N/A**.
- Shorter lines work well for **headlines**; **eyebrows** are small section labels (often lowercase).
- Many headlines include **one phrase in italics** on the site—those prompts ask for that phrase separately.
